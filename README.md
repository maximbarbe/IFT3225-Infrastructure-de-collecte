<h1>Projet</h1>
<p>Ce répo est séparé en deux parties: backend et client.</p>
<h2>Backend</h2>
<p>La partie backend contient le code nécessaire pour l'API qui fournit les données au côté client. On y retrouve le code source, les données, les scripts pour populer la base de données et les instructions sur l'utilisation de l'API.</p>
<h2>Client</h2>
<p>La partie client contient le code pour construire la couche client qui présente les données de l'API de manière intuitive et permet à des usagers d'ajouter facilement des observations. On y retrouve le code source et les instructions pour démarrer le frontend.</p>
